# Professional Rock Paper Scissors Game

A modern, high-quality take on the classic **Rock, Paper, Scissors** game. This project focuses on delivering a seamless user experience with a "Pro" visual style, built entirely using frontend web technologies.

## Features
* **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop screens.
* **Dynamic Logic:** Smart win/loss detection and automated house-play using JavaScript.
* **Score Tracking:** Real-time score updates to keep track of your performance.
* **Smooth Animations:** Polished UI with CSS transitions and interactive button effects.
* **Clean UI:** Follows a modern aesthetic with custom gradients and layout.

## Tech Stack
* **HTML5:** Structured for accessibility and semantic clarity.
* **CSS3 (Flexbox/Grid/Variables):** Used for advanced layouts and theme consistency.
* **Vanilla JavaScript (ES6+):** Handles game state, DOM manipulation, and logic without external libraries.

## How to Run
1. Clone the repository.
2. Open `index.html` in your browser.
3. Choose your weapon (Rock, Paper, or Scissors) and try to beat the house!

## Preview
Check out the `assets/images` folder to see the assets used for the clean UI.
